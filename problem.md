## 🚀 Problem Statement

Millions of people around the world rely on sign language as their primary method of communication. Unfortunately, many people do not understand sign language, creating communication barriers in daily life.

SignSpeak AI solves this problem by translating hand signs into text and speech instantly, making communication more inclusive and accessible.

---

## 💡 Solution

The platform uses Computer Vision and Artificial Intelligence to:

1. Detect hands from the camera feed.
2. Track hand landmarks.
3. Identify sign language gestures.
4. Convert recognized signs into text.
5. Convert text into speech.

This enables real-time communication between sign language users and non-sign language users.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/signspeak-ai.git
```

### Navigate to Project

```bash
cd signspeak-ai
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

Application will run on:

```bash
http://localhost:3000
```

## 🎮 How To Use

1. Open the application.
2. Allow camera access.
3. Position your hand in front of the camera.
4. Perform a sign language gesture.
5. View detected text instantly.
6. Click "Speak" to hear the translated output.
7. Use Practice Mode to improve your sign language skills.
