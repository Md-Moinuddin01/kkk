# 🤟 SignSpeak AI

> Breaking communication barriers with real-time Sign Language Recognition.

SignSpeak AI is an AI-powered web application that helps people with speech impairments communicate more easily. Using a device camera, the system detects hand signs in real time and converts them into readable text and spoken words.

The goal of this project is to create a bridge between sign language users and people who do not understand sign language.

## 🌟 Features

### 📷 Real-Time Camera Detection

* Accesses the user's camera securely.
* Detects hand gestures and sign language movements.
* Displays recognized signs instantly.

### 🔤 Sign to Text Conversion

* Converts detected signs into readable text.
* Shows live predictions on screen.

### 🔊 Text to Speech

* Converts recognized signs into voice output.
* Helps users communicate with people around them.

### 🎯 Practice Mode

* Learn and practice common sign language gestures.
* Instant feedback on correct and incorrect signs.

### 📚 Sign Library

* Collection of common signs and gestures.
* Useful for beginners learning sign language.

### 🌙 Modern User Experience

* Responsive design.
* Dark and Light mode.
* Mobile and desktop support.
* Accessibility-focused interface.

### 🔒 Privacy First

* Camera feed remains on the user's device.
* No video recording or storage.
* User data is not shared with third parties.
